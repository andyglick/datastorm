Downloading Data Storm

Data Storm requires
  • JDK 1.4 (or maybe even earlier!)
  • SWT.jar
  • A database :-)
Data Storm is a free open source data base browser. It can be found at https://sourceforge.net/project/showfiles.php?group_id=230366

Also remember to download the swt.jar from the SWT project if you don't have it already. Find it here http://www.eclipse.org/swt/ and if that't down try swt-win-3_4_rc3.jar
